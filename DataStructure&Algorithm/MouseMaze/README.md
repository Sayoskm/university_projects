# Mouse Maze

## Overview

This is a **C++/CLI WinForms application** that simulates a mouse navigating through a maze. The program demonstrates pathfinding algorithm, provides a graphical interface for visualizing the maze and the mouse's movement, and supports random maze generation using algorithm.

---

## Usage

1. Open the solution `Bn2.sln` in Visual Studio (supports CLR)。
2. Set up or load a maze layout in the GUI.
3. Place the mouse and the goal in the maze.
4. Select the desired pathfinding algorithm (if available)。
5. Click **Start** to watch the mouse find its way through the maze.

---

## Features

* **Maze Visualization** – Graphical display of the maze, mouse, and goal.
* **Random Maze Generation** – Create new mazes using maze generation algorithms (e.g., randomized DFS algorithm).
* **Pathfinding Algorithm** – Demonstrates DFS algorithm.
* **Step-by-Step Animation** – Visualizes the mouse's movement through the maze.
* **Custom Maze Editing** – Allows users to design or modify the maze layout.

---

## Algorithms

* **Depth-First Search (DFS)** – Explores as far as possible along each branch before backtracking.
* **Maze Generation Algorithms** – Randomly generate solvable mazes using DFS algorithm.
