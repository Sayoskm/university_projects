# TexasPoker

## Overview

This is a **Python-based Texas Hold'em Poker Telegram bot**. The bot allows you to play Texas Hold'em directly in Telegram chat, supporting real poker gameplay, betting rounds, and interactive commands.

---

## Usage

1. Ensure you have Python 3 installed on your system.
2. Set up your Telegram Bot Token and configure it in the code (usually in `main.py`).
3. Navigate to the `TexasPoker` directory.
4. Run the bot:
   - `python main.py`
5. Add the bot to your Telegram group or chat, and interact with it using commands.

---

## Features

* **Classic Texas Hold'em Rules** – Standard poker gameplay with betting, dealing, and hand evaluation.
* **Telegram Bot Integration** – Play poker directly in Telegram chat via bot commands.
* **Interactive Commands** – Use Telegram messages to bet, fold, check, and more.
* **Randomized Deck** – Fair shuffling and dealing every round.
* **Replayable** – Play as many rounds as you like.

---

## Project Structure

- `main.py` – Entry point and game loop
- `texasholdem.py` – Poker logic, hand evaluation
