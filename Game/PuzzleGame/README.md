
# Puzzle Game

## Overview

This is a **web-based sliding puzzle game** implemented with **HTML, CSS, and JavaScript**. The game features a classic sliding puzzle interface, coin system, store for unlocking new puzzle images, and various UI enhancements for a fun and interactive experience.

---

## Usage

1. Open `index.html` in your browser.
2. Click **Start** to begin a new puzzle.
3. Use arrow keys or on-screen buttons to move tiles and solve the puzzle.
4. Earn coins by solving puzzles efficiently.
5. Click the **Store** to unlock new puzzle images or use coins for the cheat function.
6. Use the **Settings** to toggle music and sound effects.

---

## Features

* **Sliding Puzzle Gameplay** – Classic sliding puzzle with customizable size and images.
* **Coin System** – Earn and spend coins for rewards and cheats.
* **Store** – Unlock new puzzle images with coins.
* **Cheat Function** – Instantly solve the puzzle by spending coins.
* **Timer & Move Counter** – Track your speed and efficiency.
* **Sound & Music** – Toggle background music and sound effects.
* **Responsive UI** – Works on desktop browsers.
