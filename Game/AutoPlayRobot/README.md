
# AutoPlayRobot

## Overview

This is a **Python-based Auto Play Robot** designed specifically for the online game [Wood Block Journey](https://www.crazygames.com/game/wood-block-journey) on crazygames.com. The robot simulates keyboard and mouse input to automatically play the game. This script is only compatible with the specified game and website.

---

## Usage

1. Make sure you have Python 3 installed on your system.
2. Install the `pyautogui` package:
   - `pip install pyautogui`
3. Navigate to the `AutoPlayRobot` directory.
4. Run the main script:
   - `python main.py`
5. The script will automatically open Google Chrome, navigate to the [Wood Block Journey](https://www.crazygames.com/game/wood-block-journey) game page, and start playing the game automatically.
6. Do not move your mouse or keyboard during execution to avoid interfering with the automation.

---

## Features

* **Automated Block Puzzle Gameplay** – Automation script designed specifically for Wood Block Journey.
* **Keyboard & Mouse Control** – Uses Python `pyautogui` to control mouse and keyboard.
* **Game-Specific Logic** – Optimized for the game's UI and rules.
* **One-Click Start** – Just run the script to launch and play the game automatically.
* **Replayable** – Can be used for repeated automated gameplay.
