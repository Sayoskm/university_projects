# Wordle

## Overview

This is a **web-based Wordle game** implemented with **HTML, CSS, and JavaScript**. The game challenges you to guess a hidden five-letter English word within six attempts, providing feedback for each guess.

---

## Usage

1. Open `index.html` in your browser.
2. Enter a five-letter English word and press Enter or click the submit button.
3. The game will provide feedback:
	* Green: Correct letter in the correct position
	* Yellow: Correct letter in the wrong position
	* Gray: Letter not in the word
4. You have up to six attempts to guess the correct word.
5. Play again by refreshing the page or clicking the restart button (if available).

---

## Features

* **Classic Wordle Gameplay** – Guess the hidden word in six tries.
* **Dictionary Validation** – Only valid English words are accepted (uses dictionary.txt).
* **Color Feedback** – Visual hints for each letter (green/yellow/gray).
* **Keyboard & Mouse Input** – Supports both typing and on-screen keyboard (if implemented).
* **Simple UI** – Clean and responsive interface for easy play.
* **Replayable** – Play as many rounds as you like.
